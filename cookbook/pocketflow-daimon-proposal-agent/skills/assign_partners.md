Recommend partner involvement for a work package.

Base recommendations on:
- partner expertise stated in project memory
- country/data-collection needs
- links to methods, stakeholders, and outputs
- balance across the closed consortium
- uncertainty in researcher profiles

Output:
- recommended lead
- core contributors
- optional contributors
- rationale for each role
- missing capability, if any
- risks or verification needed

Do not treat unresolved web profiles as confirmed facts. Prefer "verify this affiliation/expertise" when the memory says a profile is unresolved.
